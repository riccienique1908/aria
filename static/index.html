What's New in v4

⚠ Voice Setup Required

Voice uses browser speech until you accept Orpheus terms:

1. Go to console.groq.com/playground
2. Select canopylabs/orpheus-v1-english
3. Type any text → Submit → Accept terms

Takes 30 seconds. One-time only.

Environment Variables

Set these in Render → Environment:

GROQ_API_KEY — console.groq.com
GEMINI_API_KEY — aistudio.google.com
SUPABASE_URL — your project URL
SUPABASE_ANON_KEY — anon/public key
SUPABASE_SERVICE_KEY — service_role key
RENDER_EXTERNAL_URL — your Render URL
