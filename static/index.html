Personalise ARIA
1
Tell ARIA about yourself
The fastest way to personalise ARIA is to just tell it things about you in chat. It will remember your preferences, goals, and habits across sessions via Supabase.
2
Set your health goals
Try saying:

"My daily goal is 2000 calories and 150g protein"
"I want to work out 4 times a week"
"I'm trying to lose 10kg by summer"
3
Set up habits to track
Try saying:

"Add a habit: drink 2L of water daily"
"Add a habit: 30 min walk every morning"
"Show me my habit streaks"
4
Teach ARIA your QA style
Paste your test cases or bug reports and say:

"Here's how I write test cases — learn my style: [paste]"
"I use Playwright with TypeScript and Jira for bug tracking"
5
Fine-tune on your PC (advanced)
For deep personalisation, use the ARIA local files to fine-tune on your RTX 3060, then the model can be used as a fallback. See the README for details.
Deploy to cloud (free)
✓
Get Groq API key (free)
Sign up at console.groq.com — no credit card. Create an API key. This is your free AI brain (14,400 requests/day).
2
Create free Supabase database
Go to supabase.com → New project (free). Then run this SQL in the editor:
create table conversations (id bigserial primary key, user_message text, ai_reply text, module text, created_at timestamptz default now()); create table logs (id bigserial primary key, type text, data jsonb, created_at timestamptz default now());
3
Push code to GitHub
Upload the ARIA cloud folder to a new GitHub repo (can be private). Render will deploy directly from it.
git init && git add . && git commit -m "ARIA cloud" git remote add origin https://github.com/YOU/aria.git git push -u origin main
4
Deploy on Render (free)
Go to render.com → New → Web Service → connect your GitHub repo → set:

Build: pip install -r requirements.txt
Start: uvicorn app.main:app --host 0.0.0.0 --port $PORT
5
Set environment variables
In Render → Environment, add these:
GROQ_API_KEY=gsk_your_key_here SUPABASE_URL=https://xxx.supabase.co SUPABASE_ANON_KEY=your_anon_key APP_PASSWORD=choose_a_password RENDER_EXTERNAL_URL=https://your-app.onrender.com
6
Keep-alive with UptimeRobot
Sign up free at uptimerobot.com → Add monitor → HTTP → your Render URL + /healthz → every 5 min. This prevents Render sleeping.
7
Add to iPhone home screen
Open your Render URL in Safari → Share → Add to Home Screen → ARIA appears as an app icon. Works like a native app.
